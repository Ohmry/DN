# HTML에서 SVG 아이콘 사용 방법

1. [freeicons.io](https://freeicons.io)에 가서 사용하고 싶은 아이콘을 찾는다.
2. 원하는 아이콘을 SVG로 다운로드 받는다.
3. 다운로드 받은 파일의 소스를 HTML 안에 삽입한다.
